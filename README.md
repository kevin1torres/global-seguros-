 # Global Seguros - Rediseño Landing Page UX/UI

# Descripción del Proyecto

Este proyecto consiste en el rediseño de la landing page de Global Seguros, desarrollado como una propuesta de mejora de experiencia de usuario (UX) e interfaz de usuario (UI).

La propuesta conserva la identidad corporativa de la marca mientras incorpora una experiencia visual más moderna, emocional e inmersiva mediante elementos gráficos como un planeta, órbitas y una flor de loto que simboliza el crecimiento y las diferentes etapas de la vida.


# Objetivo

Diseñar una experiencia digital que:

* Mejore la navegación y la usabilidad.
* Refuerce la percepción de confianza y estabilidad.
* Genere una conexión emocional con el usuario.
* Presente los productos de manera más clara.
* Modernice la imagen digital de Global Seguros.
* Mantenga la línea visual corporativa de la marca.

## Customer Journey Analizado

El recorrido actual identificado en Global Seguros es:

1. Entrada a la landing page.
2. Exploración de productos.
3. Evaluación de beneficios.
4. Contacto mediante formulario.
5. Comunicación con asesor.
6. Cotización personalizada.
7. Contratación.
8. Pago.
9. Activación y gestión.

---

## Problemas Detectados

### UX

* No existe cotizador inmediato.
* Dependencia de asesores para avanzar.
* Proceso largo para convertir usuarios.
* Falta de automatización.

### UI

* Experiencia visual estática.
* Escasa profundidad gráfica.
* Baja carga emocional.
* Poca interacción visual.

---

## Solución Propuesta

La nueva landing incorpora:

* Hero principal inmersivo.
* Storytelling visual.
* Sistema de crecimiento progresivo de una flor de loto.
* Mayor jerarquía visual.
* CTAs más visibles.
* Mejor organización de la información.
* Animaciones suaves.
* Diseño responsive.
